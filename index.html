    This website hosts the Standard Operating Procedures (SOPs) for common data analysis tasks within the H3ABioNet consortium.

    The following have associated node accreditation exercises:

    • Human Variant calling
    • Genome Wide Association Studies
    • Gene expression analysis using RNA-Seq
    • 16s microbiome analysis

    In addition, we have additional SOPs that cover some of the reporting and review process:

    • Reviewers of node accreditation reports

    These SOPs are meant to help various H3ABioNet nodes in their accreditation exercises 1, as per this protocol.

    You can use the buttons at the top of the page to navigate those various SOPs, or the side panel to access the subsections within each.

    Feedback is also welcome! Please feel free to fork this repo, open an issue or a pull request. Additionally, check out our contribution guide.
